# How to cite

If you use this code or build on the paper, please cite the current preprint:

## BibTeX

```bibtex
@misc{anjum2026sentinel,
  author       = {Muneeb Anjum},
  title        = {SENTINEL-LLM: A Practical Multi-Layer Defense Framework for Adversarial Prompts in Production Language Model Systems},
  year         = {2026},
  howpublished = {Preprints.org},
  note         = {Companion code: \url{https://github.com/horizonbymuneeb/sentinel-llm}}
}
```

## APA

Anjum, M. (2026). *SENTINEL-LLM: A Practical Multi-Layer Defense Framework for
Adversarial Prompts in Production Language Model Systems*. Preprints.org.

## IEEE

M. Anjum, "SENTINEL-LLM: A Practical Multi-Layer Defense Framework for
Adversarial Prompts in Production Language Model Systems," Preprints.org, 2026.

## Plain text

Muneeb Anjum (2026). SENTINEL-LLM: A Practical Multi-Layer Defense Framework
for Adversarial Prompts in Production Language Model Systems. Preprints.org.
